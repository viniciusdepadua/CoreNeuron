18 /**
19  * \class corenrn_parameters
20  * \brief Parses and contains Command Line parameters for Core Neuron
21  *
22  * This structure contains all the parameters that CoreNeuron fetches
23  * from the Command Line. It uses the CLI11 libraries to parse these parameters
24  * and saves them in an internal public structure. Each parameter can be
25  * accessed or written freely. By default the constructor instantiates a
26  * CLI11 object and initializes it for CoreNeuron use.
27  * This object is freely accessible from any point of the program.
28  * An ostream method is also provided to print out all the parameters that
29  * CLI11 parse.
30  * Please keep in mind that, due to the nature of the subcommands in CLI11,
31  * the command line parameters for subcategories NEED to be come before the relative
32  * parameter. e.g. --mpi --gpu gpu --nwarp
33  * Also single dash long options are not supported anymore (-mpi -> --mpi).
34  */
